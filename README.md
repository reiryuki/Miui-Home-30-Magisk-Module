# Mi Wallpaper Carousel Magisk Module

## Descriptions
- Mi Wallpaper Carousel app by Xiaomi Inc. ported for any Android 5 and up non-Miui ROM.

## Requirements
- Not in Miui ROM
- Android 5 and up
- Magisk installed

## Installation Guide
- Install [Miui Core Magisk Module](https://github.com/reiryuki/Miui-Core-Magisk-Module) first. Tap here: https://github.com/reiryuki/Miui-Core-Magisk-Module
- Install this module via Magisk app or Recovery
- Reboot
